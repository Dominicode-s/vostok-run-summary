# Run Summary

**After-action report for every raid in Road to Vostok.**

Run Summary shows a detailed breakdown of your run whenever you die or return to the shelter — what you killed, what hurt you, what you looted, and how your survival stats changed along the way.

## Features

- **Automatic post-run modal** — appears after death or shelter return
- **Combat** — kills and damage taken
- **Loot** — items picked up during the run
- **Survival** — energy, hydration, health, and mental drain + restoration from eating/drinking
- **Economy** — cash earned (with [Cash System](https://github.com/Dominicode-s/vostok-cash))
- **XP** — experience gained (with [XP & Skills System](https://github.com/Dominicode-s/vostok-skills))
- **Run timer** — how long you survived
- **History** — browse your last 10 runs
- **F6 hotkey** — reopen the last summary anytime (configurable via MCM)
- **Zero conflicts** — pure autoload mod, no script overrides

## Installation

1. Download `Run-Summary.vmz` from [Releases](https://github.com/Dominicode-s/vostok-run-summary/releases)
2. Drop it into your `Road to Vostok/mods/` folder
3. Launch the game

## Optional Dependencies

These mods are not required but Run Summary will show additional stats when they're installed:

- [Cash System](https://github.com/Dominicode-s/vostok-cash) — cash earned per run
- [XP & Skills System](https://github.com/Dominicode-s/vostok-skills) — XP gained per run

## Configuration (MCM)

If you have [Mod Configuration Menu](https://modworkshop.net/mod/48028) installed:

| Setting | Default | Description |
|---------|---------|-------------|
| Auto-Show Summary | On | Automatically show the modal after each run |
| Reopen Hotkey | F6 | Key to reopen the last run summary |
